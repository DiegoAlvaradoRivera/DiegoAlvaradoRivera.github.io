Bienvenido

Esta página es un portafolio de mis projectos de ciencia de datos

Real State Prediction
  • Objetivo: predecir el precio por unidad de área de los inmobiliarios e identificar cómo las diferentes variables influyen en el precio.
  • Dataset: para este proyecto se utilizó el Real state valuation data set, el cual contiene información histórica de valuaciones inmobiliarias realizadas en Sindian District, New Taipei City, Taiwan en los años 2012 y 2013.
  • Técnicas: GAM, Random Forests
ENARES agrupamiento
  • Objetivo: encontrar perfiles de tolerancia hacia la violencia e identificar la relación con otros factores como sexo, educación, etc.
  • Dataset: para este proyecto se utilizó la Encuesta nacional sobre relaciones sociales 2019 dirigida a varones y mujeres de 18 y más años de edad
  • Técnicas: k-means
